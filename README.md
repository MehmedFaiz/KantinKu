# KantinKu

KantinKu adalah proyek antarmuka aplikasi kantin berbasis HTML, CSS, dan JavaScript yang dirancang untuk membantu pengelolaan kantin secara digital. Proyek ini memiliki berbagai tampilan dashboard dan fitur navigasi untuk admin maupun pengguna.

## Cara Menjalankan Proyek

1. Download atau clone repository KantinKu.
2. Buka folder proyek menggunakan Visual Studio Code atau file explorer biasa.
3. Jalankan file `index.html` pada browser.
4. Halaman awal akan menampilkan fitur Login.
5. Pengguna dapat:
   - Login menggunakan email dan password
   - Masuk ke halaman Register jika belum memiliki akun
6. Setelah login berhasil, pengguna akan diarahkan ke Dashboard Admin.
7. Di dalam Dashboard Admin tersedia beberapa menu navigasi menuju halaman lain seperti:
   - Profile
   - Papan Status
   - Operational Penjualan
   - Dashboard lainnya

Proyek dapat dijalankan langsung tanpa instalasi tambahan karena menggunakan HTML, CSS, dan JavaScript murni.
